## How to Run

* Install dependencies
```bash
npm install
```

* Start the dev server
```bash
npm run dev
```

**The server will be available at http://localhost:3000**

* Build for production
```bash
npm run build
```

* Run the production build
```bash
npm start
```

## Dependencies

* Node.js 18+
* Express
* TypeScript
* ts-node
* nodemon
